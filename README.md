# Marketing AI Acceleration Playbook

A practical reference of AI-powered workflows across the full marketing lifecycle - from predictive strategy through brand-safe execution.

## What This Is

An interactive table mapping 13 marketing workflows to specific AI solutions, tool stacks, and success metrics. Each workflow is categorised by marketing function (Strategy, Research, Content, Campaign, Demand Gen, Analytics, Governance) and can be filtered in the live page.

## Workflows Covered

- **Strategy & Planning** - Campaign brief generation, audience segmentation, predictive intent
- **Research & Intel** - Automated competitive monitoring and signal detection
- **Content Creation** - Text content multiplication, visual/video pipeline (Figma, CapCut, Midjourney), dynamic SEO
- **Campaign Execution** - Segment-level email personalisation, ad copy factory, champion-challenger A/B framework
- **Demand Generation** - Behaviour-triggered lead nurture sequences
- **Analytics & Reporting** - Closed-loop performance digest with feedback to strategy prompts
- **Governance & Safety** - Human-in-the-loop checkpoints and PII scrubbing

## Tool Stack Architecture

Each workflow uses a consistent four-layer stack:

| Layer | Role | Examples |
|-------|------|----------|
| **Brain** | AI reasoning | Claude API, Perplexity, Midjourney |
| **Interface** | Where output lands | Slack, Notion, Figma, HubSpot |
| **Connector** | Automation glue | Zapier |
| **Data** | Input source | CRM exports, GA4, brand guidelines |
